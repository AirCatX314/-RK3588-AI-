# 实验室安全监控系统 - Qt客户端 + 服务器架构

## 系统架构

```
┌─────────────────┐       ┌─────────────────┐       ┌─────────────────┐
│  IP摄像头       │       │  开发板(Qt界面) │       │  管理员电脑     │
│                 │       │                 │       │                 │
│  RTSP流         │──────▶│  实时监控      │──────▶│  接收报警      │
│                 │       │  控制面板       │       │  消息记录      │
│                 │       │  紧急求助       │       │  远程查看      │
└─────────────────┘       └─────────────────┘       └─────────────────┘
                                   │
                                   ▼
                          ┌─────────────────┐
                          │  消息服务器      │
                          │  (WebSocket)    │
                          │                 │
                          │  报警推送       │
                          │  消息转发       │
                          └─────────────────┘
```

## 模块说明

### 1. 消息服务器 (lab_msg_server.py)
- WebSocket服务器
- 管理客户端连接
- 消息广播/转发
- 报警推送

### 2. Qt工控客户端 (lab_client.py)
- PyQt5界面
- 实时视频监控
- 报警按钮
- 消息发送

### 3. 管理员端 (admin_client.py)
- 接收报警
- 发送通知
- 日志记录

## 通信协议

```json
{
  "type": "alert",        //消息类型: alert, normal, emergency, status
  "level": "danger",      //级别: normal, warning, danger
  "title": "火灾报警",
  "content": "检测到火焰",
  "timestamp": "2026-03-16 10:00:00",
  "sender": "lab-device-1"
}
```
